# 🛡️ Manager-Money v1.3.6
### מערכת ניהול תקציב חודשי חכמה, מהירה ובעיצוב Glassmorphism עתידני

**Manager-Money** היא אפליקציית ווב (Web App) מתקדמת, המעניקה לך שליטה מלאה על התזרים הפיננסי שלך (הכנסות והוצאות). המערכת נבנתה עם דגש על שילוב של **עיצוב מודרני ("מגן הנתונים הזוהר")** עם ביצועים מהירים ואבטחת נתונים מקומית.

---

## ✨ תכונות ליבה (Core Features)

* **מיתוג "מגן הנתונים" (Glowing Data Shield):** לוגו מגן אלגנטי עם גרף עמודות פנימי המבצע אנימציית פעימה (Pulse) עדינה, המסמלת ביטחון וצמיחה פיננסית.
* **תצוגת תאריכים מדויקת (Date Precision):** תאריך קטן (יום/חודש) מוצג מתחת לתיאור כל פעולה לסדר מקסימלי.
* **ניתוח ויזואלי בזמן אמת:**
    * **גרף פלחים (Chart.js):** Doughnut Chart דינמי המציג חלוקה באחוזים של הכנסות, הוצאות והוצאות קבועות.
    * **מד ניצול תקציב (Progress Bar):** מד אנימטיבי המשתנה מצבע ירוק לאדום לפי אחוז הבזבוז מההכנסה.
* **חוויית משתמש (UX) משופרת:**
    * **מצב עריכה בולט:** כפתור השמירה משנה צבע לכתום בזמן עריכת פעולה קיימת.
    * **אנימציות חלקות:** אנימציות Slide-in חלקה של פריטים לרשימה ואנימציות כניסה של הגרף.
* **אבטחת נתונים מקומית:** המערכת עובדת Offline-first ושומרת את כל המידע בדפדפן שלך (`localStorage`).

---

## 💾 גיבוי, ייצוא ומיזוג (Data Management)

כדי להבטיח שהמידע שלך לעולם לא יאבד, הוספנו כלי ייצוא ומיזוג חכמים:

1.  **💾 גיבוי (JSON):** שומר את כל מסד הנתונים לקובץ גיבוי שניתן לשלוח למכשיר אחר.
2.  **📊 אקסל (CSV):** מייצא את הנתונים לפורמט טבלאי לצורך ניתוח מעמיק ב-Excel או Google Sheets.
3.  **🔄 מיזוג חכם (Merge):** מאפשר לטעון קובץ גיבוי חיצוני ולמזג אותו עם הנתונים הקיימים ללא כפילויות.

---

## 🚀 הוראות התקנה ושימוש

1.  שמור את הקוד כקובץ `money.html`.
2.  פתח את הקובץ בכל דפדפן מודרני (Chrome, Edge, Safari).
3.  **התקנה כאפליקציה (PWA):** בדפדפן כרום/אדג', לחץ על "שלוש הנקודות" > "שמור ושתף" > "התקן דף כאפליקציה".
