# 🛡️ Manager-Money 2026

**Manager-Money** היא אפליקציית ווב (Web App) מתקדמת לניהול תקציב אישי, המעוצבת בסגנון **Glassmorphism** מודרני. האפליקציה מאפשרת מעקב מדויק אחרי הכנסות, הוצאות קבועות ומשתנות, עם תצוגה ויזואלית חכמה של ניצול התקציב.

---

## ✨ פיצ'רים עיקריים

* **📊 תצוגה ויזואלית:** גרף עוגה (Doughnut Chart) אינטראקטיבי המציג את חלוקת הכספים בזמן אמת.
* **📅 ניווט חודשי:** אפשרות לעבור בין חודשים ולראות היסטוריה מלאה של פעולות.
* **🔍 סינון חכם:** פילטרים מהירים להצגת "הכל", "הכנסות" או "הוצאות" בלבד.
* **💾 ניהול דאטה:** * גיבוי מלא לקובץ JSON.
    * ייצוא נתונים ל-Excel (CSV) לצורך ניתוח חיצוני.
    * **מיזוג חכם:** אפשרות לייבא נתונים ממכשירים אחרים ללא דריסת מידע קיים.
* **📱 מותאם לנייד:** ממשק Responsive מלא עם חוויית משתמש חלקה ב-iOS ובאנדרואיד.
* **🛡️ אבטחה ופרטיות:** כל הנתונים נשמרים מקומית בדפדפן (LocalStorage) - המידע שלך נשאר אצלך.

---

## 🎨 עיצוב וממשק

האפליקציה משתמשת בשפת עיצוב **Glassmorphism** הכוללת:
* רקעים מדורגים (Radial Gradients).
* אפקט טשטוש (Backdrop Filter) על כרטיסי המידע.
* **לוגו יצירתי:** מגן וגרף מונפש המסמלים ביטחון וצמיחה פיננסית.

---

## 🛠️ טכנולוגיות

* **Frontend:** HTML5, CSS3 (Flexbox/Grid), JavaScript (Vanilla).
* **Charts:** [Chart.js](https://www.chartjs.org/) להצגת נתונים ויזואליים.
* **Fonts:** Heebo (Google Fonts) לתמיכה מושלמת בעברית.
