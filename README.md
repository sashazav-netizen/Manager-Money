# 💰 Manager-Money 2026 Pro (v3.1.4.6)

**Manager-Money** היא אפליקציית ווב (Web App) מתקדמת לניהול תקציב אישי, שנועדה לספק שליטה מלאה על הכנסות והוצאות בממשק מודרני, מהיר וחכם. האפליקציה מותאמת אישית לשימוש במובייל וכוללת סנכרון ענן בזמן אמת.

---

## ✨ תכונות עיקריות

* **ניהול תזרים מזומנים:** מעקב אחרי הכנסות, הוצאות רגילות והוצאות קבועות.
* **ניתוח גרפי חכם:** גרף דונאט (Doughnut Chart) המציג פילוח תקציבי ושורת **יתרה פנויה** (Net) לחישוב הרווח החודשי.
* **סנכרון ענן (Firebase):** גיבוי וסנכרון נתונים אוטומטי המאפשר גישה מכל מכשיר.
* **מצב פרטיות (Privacy Mode):** אפשרות לטשטוש סכומים רגישים בלחיצת כפתור.
* **ניהול הוצאות קבועות:** כפתור ייעודי לשכפול הוצאות קבועות מחודשים קודמים לחודש הנוכחי.
* **ייצוא נתונים:** אפשרות לייצוא הנתונים לקובץ **CSV** (אקסל) או לגיבוי מלא בפורמט **JSON**.
* **מותאם למובייל:** עיצוב רספונסיבי מלא (PWA Ready) עם חוויית משתמש חלקה ב-iOS ובאנדרואיד.

---

## 🛠 טכנולוגיות

* **Frontend:** HTML5, CSS3 (Glassmorphism UI), JavaScript (ES6+).
* **Charts:** [Chart.js](https://www.chartjs.org/) לניתוח נתונים ויזואלי.
* **Backend:** [Firebase Realtime Database](https://firebase.google.com/) לאחסון וסנכרון.
* **Authentication:** Firebase Anonymous Auth לזיהוי ייחודי ללא צורך בהרשמה מסורבלת.
* **Fonts:** Google Fonts (Heebo & Assistant).

---

## 🚀 הוראות הפעלה

1.  הורד את הקובץ `index.html`.
2.  פתח את הקובץ בכל דפדפן מודרני (Chrome, Safari, Edge).
3.  **שימוש בנייד:** מומלץ להוסיף את הדף ל"מסך הבית" (Add to Home Screen) כדי לקבל חוויה של אפליקציה מובנית.

---

## 📂 מבנה הנתונים

הנתונים נשמרים באופן מקומי ב-`localStorage` לסנכרון מיידי, ובענן תחת מזהה משתמש אנונימי מאובטח.

**שדות הנתונים:**
- `id`: מזהה ייחודי (Timestamp).
- `desc`: תיאור הפעולה.
- `amount`: סכום הפעולה.
- `type`: סוג (income/expense/fixed).
- `date`: תאריך הביצוע.
- `done`: האם הפעולה בוצעה (V) - משפיע על יתרת העו"ש.

---

## 🔒 פרטיות ואבטחה

האפליקציה משתמשת באימות אנונימי של Firebase. הנתונים שלך משויכים למכשיר שלך בלבד ולא נחשפים למשתמשים אחרים. מומלץ לבצע ייצוא לגיבוי (JSON) מדי פעם.
